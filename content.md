# ¿Por qué Python para Ciencia?

* Sintaxis sencilla (fácil de leer y escribir)
* Multiplataforma
* Código abierto: _**libertad**_ para utilizarlo como queremos
* Múltiples librerías científicas
* **Comunidad** científica y de desarrolladores
